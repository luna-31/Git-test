## 1. Git
파일 변경 이력을 관리하는 도구

## 2. GitHub
쉽게 말하면 파일을 저장하고 관리할 수 있는 온라인 저장소?

## 3. Clone
GitHub의 Repository를 VSCode로 가져오기

## 4. Commit
파일 저장하기. 게임의 savepoint 느낌

## 5. Push
VSCode에서 작업한 파일을 GitHub에 업로드

## 6-1. Branch
(=나뭇가지) Main이라는 중심 가지가 있고 그 위에 Branch라는 가지가 있음. Branch에서 각각 개인의 파일을 만들고 그 파일이 제대로 작동이 된다고 판단하면 Main Branch로 이동

## 6-2. pull request
(=코드 병합) 6-1에서 말한 Main Branch로 이동을 뜻함.
